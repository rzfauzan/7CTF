## 😹 About

7CTF adalah mini CTF di mana kamu akan berhadapan dengan:

- file yang katanya "sudah aman"
- PDF yang ngomong sendiri
- gambar gorilla yang menyimpan trauma
- log server dari dimensi lain
- anak magang yang sudah menyerah duluan

Semua challenge dibuat dengan konsep sederhana:

> "hal sederhana yang jadi penderitaan digital."
